Skateboard Shop Site Plan

Site Name

Name: SkateCraft

Reason: The name "SkateCraft" was chosen because it combines "Skate" for skateboarding and "Craft" to emphasize the customization and craftsmanship of building your own skateboard.

Site Purpose

The site provides an online skateboard shop offering various skateboard parts such as decks, trucks, wheels, and bearings. It features an interactive tool that allows users to build and customize their own skateboard by combining different parts before purchasing. Additionally, the site offers detailed product information, user reviews, and guidance on selecting parts for beginners and advanced riders alike.

Scenarios

  • Which skateboard deck is best suited for street skating?
  • How can I customize my skateboard to improve its speed and durability?

Color Schema

  • #A8E10C - accent color for some texts
  • #1A1A1D - main background color for website
  • #E94560 - secondary background color for some headings
  • #F0F0F0 - main text color
  • #FFFFFF - secondary text color for some headings

Typography
